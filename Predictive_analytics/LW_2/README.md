# Виртуальный датчик для контроля процесса обжига в печи

## Описание проекта
Проект посвящён построению виртуального датчика, который прогнозирует концентрацию выходного продукта в текущий момент времени на основе минутной телеметрии печи.

## Что делает решение
- синхронизирует телеметрию и лабораторные замеры с учётом задержки;
- создаёт временные признаки;
- обучает и сравнивает три модели;
- оценивает качество по нескольким метрикам;
- анализирует остатки и важность признаков;
- имитирует работу виртуального датчика в реальном времени;
- отслеживает drift данных.

## Структура решения
1. EDA и синхронизация данных
2. Feature engineering
3. Обучение моделей
4. Оценка качества
5. Интерпретация и real-time сценарий

## Запуск
1. Установить зависимости из requirements.txt
2. Поместить csv-файлы в рабочую директорию
3. Запустить notebook последовательно сверху вниз

## Бизнес-результат
Решение позволяет сократить задержку получения информации о качестве продукта и повысить оперативность управления печью.
